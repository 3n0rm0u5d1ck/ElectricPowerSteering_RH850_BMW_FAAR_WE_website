-   **<u>May</u>**: indicates an optional action.

# Effort & High-Level Description

This function produces a handwheel reference torque for the closed loop
system. The function should generate torque equal to the effort felt the
driver on the handwheel. Output effort reference torque is a function of
the Rack Force Estimated and Vehicle Speed.
